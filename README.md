# actividad-sesi-n-4
La página de Sabores Urbanos es un sitio de restaurante de alta cocina que presenta una experiencia gastronómica elegante y moderna. Incluye un menú gourmet con platillos exclusivos, imágenes atractivas, una sección para reservas, información del restaurante y datos de contacto todo con un diseño sofisticado enfocado en transmitir lujo y calidad
